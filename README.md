# 🚀 AI_Playground

A full-stack MERN application where users can **learn, explore AI tools, solve problems, and have fun** — all in one place.

---

## 🌟 Features

### 🎉 Fun Zone

* Music 🎵
* Jokes 😂
* Quotes 💬

### 📚 Study Hub

* Coding guides
* Problem solving
* Learning paths

### 🤖 AI Tools

* Mini AI-based tools
* Experiment with AI concepts

### 🧪 Project Lab

* Build and explore small projects

### 👤 User System

* Register & Login
* Profile with image upload
* Secure authentication

---

## 🛠 Tech Stack

### Frontend

* React ⚛️
* Tailwind CSS 🎨
* Framer Motion ✨
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB 🍃
